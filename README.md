# Speed Typing Test

![Скриншот приложения](screenshot.png)

Простое приложение для проверки скорости печати на Python с графическим интерфейсом.

## Возможности
- Случайная генерация текста
- Таймер запускается при первом нажатии клавиши
- Подсчёт скорости и точности печати
- Визуализация ошибок
- Серый дизайн с двумя оттенками

## Как запустить
1. Установи Python 3.x
2. Скачай файл `speedtest.py`
3. Запусти командой: `python speedtest.py`

## Требования
- Python 3.x
- tkinter (обычно уже установлен)
